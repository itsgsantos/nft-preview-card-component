## 🚀 Technical Features

* **Interactive Overlay:** Implementation of a hover state on the main image with a color overlay and eye icon.
* **Semantic HTML:** Structure built with accessibility in mind using proper container tags.
* **Modern CSS Layout:** Utilization of Flexbox for vertical and horizontal alignment of card elements and icons.
* **Custom Properties:** CSS Variables for managing the specific color palette (Cyan, Soft Blue, etc.) and typography.
* **Responsive Design:** Fully optimized for mobile and desktop screens following the professional design spec.

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** CSS3 (Flexbox, Overlay effects, Transitions)
- **Design System:** Provided by Frontend Mentor

## 🧠 Key Learnings

Building this NFT component was an excellent exercise in handling interactive states and precise layout alignments.

**Main Takeaways:**
- Creating a **custom overlay effect** using absolute positioning and opacity transitions.
- Aligning icons (Ethereum and Clock) perfectly with text using **Flexbox inline-gap**.
- Managing transparency and color blending to match the specific design requirements.
- Refining the use of **border-radius** and box-shadows to create a modern "card" aesthetic.
